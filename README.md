# 📱 Pocket Run — Run Code on Your Phone

Run **Python, JavaScript, Web projects** directly on your Android phone. No laptop required. Upload a folder or ZIP and get live output + full errors.

A beautiful, installable PWA built for developers on the go.

## ✨ Features

- Run Python code (via Pyodide)
- Execute JS and simple web projects
- Upload folder or ZIP file
- Live console output with errors
- Fully offline capable after install
- Beautiful mobile-first UI
- Installable as PWA / APK

## 🚀 How to Use

1. Open `index.html` in Chrome on Android
2. Install as PWA (Add to Home Screen)
3. Upload your project folder or ZIP
4. Run and see instant output

## 🛠️ Tech

- HTML + CSS + JavaScript
- Pyodide for Python execution in browser
- PWA ready

## ⚠️ Limitations

- Large projects or heavy computation may be slow
- Some Python libraries not supported in Pyodide

Perfect for quick tests, learning, and development without a computer nearby.
